MOUSE EVENTS You can change color of things when mouse moves over them. KEYBOARD EVENTS (keydown, keypress and keyup) Shift, control, alt and meta will fire the keydown and keyup events, but no the keypress event. TOUCH EVENTS Smart phones and touchscreens. touchstart event happens when user touches the surface. touchmove- when user touches screen and moves around without leaving. touchenter- user already started touching surface, then passes over the element to which the event listener is attached. touchleave - still touching surface, but leaves element to which event listener is attached. touchcancel - when touch event is interrupted.
